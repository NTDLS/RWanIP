Configure



RWanIP Configuration Dialog

Overview:

To configure RWanIP, you will either need to open the configuration utility from the installed start menu group. Also, if the application is running you can also right click on the installed "tray icon" and select configure from the popup menu. You can modify all of the application functionality from this configuration dialog, such as connection interval, FTP server (if required) and local file name.

"IP File" Section:

The IP file section of the configuration dialog allows you to specify:
  • Obtain WAN IP from
    • In order to discover what your external WAN IP address is, RWanIP must make a request to an external source to inform it of your IP address. This is where you can tell RWanIP to retrieve your IP from.
  • Save IP file location
    • When RWanIP makes the external request to retrieve your IP information, the server is going to return an HTML or XML file. This returned file will be downloaded and stored in this location.
  • Update Interval
    • You can configure to check for IP changes at an interval between 10 and 15,000 minutes.

"FTP Upload" Section:

You can configure RWanIP to upload your IP information to an FTP server. This was developed to allow you to be able to grab your current home or office IP address from a known FTP server when you are offsite.

Note: This has proven especially useful for companies that have many customers who they support remotely, as they can install RWanIP and consolidate all of their customers IP addresses into one FTP location.
  • Server
    • This should be set to the host name or IP address of an FTP server of which you have write privileges.
  • Remote Path & Filename
    • This should be set to the file name that you wish to save the IP information to on the FTP server, such as "MyCustomer.html" (without quotes). You can also specify a remote path such as "/Customers/MyCustomer.html" (without quotes) to put the file in a specific directory/folder.
  • Username
    • The FTP account username. This is used to login to the FTP server.
  • Password
    • The FTP account password. This is used to login to the FTP server.
    • Security Note: This password is stored encrypted in binary format within the registry. While it would be very difficult to obtain the IP from the registry due to encryption, it is still quite easy to grab the password from the TCP/IP traffic in the FTP login process. This is where a very good FTP server security scheme comes in handy.
  • Use Passive FTP Transfer?
    • Passive FTP is a more secure form of data transfer in which the flow of data is set up and initiated by the FTP client (RWanIP) rather than by the FTP server.
    • When unchecked, RWanIP will use an Active Mode FTP transfer.
    • If you are having trouble uploading to the FTP server, try un-checking this box.
  • Upload IP file to FTP Server?
    • This check box simply enables / disables the FTP upload process.

"Statistics" Section:

The Statistics section of the configuration dialog shows you some of the connection statistics as well as the basics connection status.
  • Last update
    • This is the local time of the last IP check (and FTP upload if applicable).
  • Last IP address
    • This is the last IP address known to RWanIP at the time of the last successful update.
  • Last Status
    • This is the status of the last attempt to check your IP address.
