Windows Service



Overview
A "Windows service" is a long-running executable that performs specific functions and which is designed not to require user intervention. Usually, Windows services start when Microsoft Windows is booted and run in the background as long as Windows is running.

How do I use it?
Well, thats the great thing about a windows service. You really don't have to do anything. The service will start when you turn your computer on even when no users are logged into your computer.

Changing from Service to Standalone and visaversa
RWanIP Setup at the Automatic Startup Step.

RWanIP's setup package has several automatic-startup options.

When you first installed RWanIP, you were presented with several automatic-startup options.
  • Install as service? (Recommended)
    • This option installs the application as a windows service. This is the superior option and requires much less user intervention.
    • Highly reccomended configuration.
    • Not available in windows 95/98.
  • Run at Startup?
    • This option allows you to run RWanIP only when somone is logged into the computer. You can specifiy if it will run only when you are logged in, or if it will run regardless of the logged in user.
  • Do not auto-start?
    • This option installs RWanIP with no auto-start option. In order to start RWanIP you would have to navigate through the installed start menu and click on the RWanIP icon.
