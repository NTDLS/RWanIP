The System Tray



Overview
The system tray is actually the name of that location (is usually in the lower right corner of your screen) that contains small icons. This allows you to easily access the features of RWanIP since it is always running in the background anyway. The tray icon also allows RWanIP to let you know when an IP update doesn't go as planed.

RWanIP Success Tray Icon with visible ToolTip.

This image (Looks like a chain-link) shows that the last IP check was successful. If you place your mouse cursor over it, it will display the IP address and the date & time it was obtained.

RWanIP Error Tray Icon with visible ToolTip.

This image (Looks like a "broken" chain-link) shows that the last IP check was NOT successful. If you place your mouse cursor over it, it will display error information. You can goto the configuration dialog to see more information and to correct the problem.

How do I use it?
You can use the Tray Icon to easily view your current IP and to copy it to the clipboard with just a click or two.

RWanIP Error Tray Icon with visible Menu.

If you right-click on the RWanIP tray icon, you will be presented with this menu.

The IP file section of the configuration dialog allows you to specify:
  • Configuration
    • Opens the configuration dialog.
  • Help
    • Opens and displays this help file.
  • Update Now
    • Attempts to imeadiatly resolve your IP address, overriding the scheduled duration.
  • Copy IP to Clipboard
    • Copies the most recently obtained IP address to your clipboard for easy use elsewhere.
  • Cancel
    • Closes the RWanIP menu, without performing any actions.
  • Exit / Shutdown
    • Completely exits the RWanIP application.
    • Displays as "Shutdown" when running as a Windows Service.
